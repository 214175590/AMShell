# AMShell
用c#开发的仿XShell客户端，集成了Springboot项目、tomcat、nginx、ice项目的监控，一键部署、启动、停止等功能，还能设置自动任务、执行自定义脚本、按条件监控项目执行脚本等等有用功能，为开发团队带来方便，您可以自行扩展功能，想咋整就咋整.
工程使用Visual Studio 2013开发。
