从服务器下载文件到本地：
[root@localhost ~]$ get [remote file path] [local file path]

上传本地文件到服务器：
[root@localhost ~]$ put [local file path] [remote file path]

更改服务器目录到“路径”：
[root@localhost ~]$ cd [path]

更改文件“路径”的权限“模式”：
[root@localhost ~]$ chmod mode [path]

更改文件“路径”的“拥有者”：
[root@localhost ~]$ chown own [path]

显示服务器“路径”下的文件列表：
[root@localhost ~]$ ls [path]

在服务器路径下创建“目录”：
[root@localhost ~]$ mkdir [path]

删除服务器路径下的“目录”：
[root@localhost ~]$ rmdir [path]

修改服务器的目录名称：
[root@localhost ~]$ rename [oldname] [newname]

显示当前目录路径
[root@localhost ~]$ pwd

显示全部帮助命令
[root@localhost ~]$ help 或者 ?
