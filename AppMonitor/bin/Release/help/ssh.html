显示当前目录路径
[root@localhost ~]$ pwd

更改服务器目录到“路径”：
[root@localhost ~]$ cd [path]

在服务器路径下创建“目录”：
[root@localhost ~]$ mkdir [path]

删除服务器文件或目录：
[root@localhost ~]$ rm [path]
[root@localhost ~]$ rm -rf [path]
[root@localhost ~]$ rm --help

重命名或移动文件/目录：
[root@localhost ~]$ mv [path] [new path]
[root@localhost ~]$ mv [name] [new name]

端口占用查询：
[root@localhost ~]$ netstat –apn | grep [port]
[root@localhost ~]$ lsof -i:[port]

新增用户：
[root@localhost ~]$ useradd [acc]
[root@localhost ~]$ passwd [acc]

安装zip 、unzip 命令组件
[root@localhost ~]$ yum install -y unzip zip

把文件授权给当前用户
[root@localhost ~]$ chmod +x [file]

文件权限转移
[root@localhost ~]$ chown [username]:[username] [file]
[root@localhost ~]$ chown -R [username]:[username] [file]

关闭Conter Os系统防火墙
[root@localhost ~]$ systemctl stop firewalld.service

快速开放端口（重启后失效）：
[root@localhost ~]$ iptables -I INPUT -p tcp --dport [port] -j ACCEPT

解压：
//解压 tar包
[root@localhost ~]$ tar –xvf [*.tar]
//解压tar.gz
[root@localhost ~]$ tar -xzvf [*.tar.gz]
//解压 tar.bz2
[root@localhost ~]$ tar -xjvf [*.tar.bz2]
//解压tar.Z
[root@localhost ~]$ tar –xZvf [*.tar.Z]
//解压rar
[root@localhost ~]$ unrar e [*.rar]
//解压zip
[root@localhost ~]$ unzip [*.zip]

1、*.tar 用tar –xvf 解压
2、*.gz 用gzip -d或者gunzip 解压
3、*.tar.gz和*.tgz 用tar –xzf 解压
4、*.bz2 用bzip2 -d或者用bunzip2 解压
5、*.tar.bz2 用tar –xjf 解压
6、*.Z 用uncompress 解压
7、*.tar.Z 用tar –xZf 解压
8、*.rar 用unrar e 解压
9、*.zip 用unzip 解压
